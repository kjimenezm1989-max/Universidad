## 📊 Objetivos del análisis

- Explorar la estructura del dataset
- Identificar valores faltantes
- Realizar análisis estadístico descriptivo
- Analizar relaciones entre variables
- Visualizar datos mediante gráficos
- Interpretar resultados

---

## 📈 Técnicas utilizadas

### 🔹 Estadística descriptiva
- Media
- Mediana
- Desviación estándar
- Cuartiles
- Rango intercuartílico

### 🔹 Análisis exploratorio (EDA)
- Distribución de variables
- Identificación de valores nulos
- Detección de outliers

### 🔹 Visualización de datos
- Histogramas
- Boxplots
- Heatmap de correlación
- Gráficos de dispersión

---

## 🧠 Variable objetivo

La variable objetivo del análisis es:

- `target`

Esta variable representa el valor principal a estudiar en relación con las demás variables del dataset.

---

## 📉 Principales hallazgos

- El dataset presenta valores faltantes en varias variables.
- Existen variables con mayor correlación con la variable objetivo.
- Se observan diferencias en la distribución de las variables numéricas.
- Se identifican posibles relaciones entre variables explicativas y el target.
